### 들어가며
온보딩 일정과, 주차별로 어떤 것을 할 예정인지 알아봐요.

### 온보딩 일정
온보딩 절차는 총 4주로 구성되어 있어요.
|주차|설명|
|--|----|
| 1주차 | 오리엔테이션 및 개발 환경 구축 |
| 2주차 | 툴 사용법 익히기: 소프트웨어, 회로 |
| 3주차 | 툴 사용법 익히기: 회로, 캐드 및 3D 프린팅 |
| 4주차 | 미니 프로젝트 |

온보딩 일정이 완료된 이후에는 곧바로 프로젝트에 참여하게 될 예정이에요. \
잘 따라온다면 어렵지 않게 적응할 수 있을거에요.

### 1주차
지금 여러분이 진행중인 단계에요. \
앞으로 4주 간 어떤 것을 배우게 될지, 개발 문화는 무엇인지 알아보게 돼요.

또, 개발 환경을 구축할 예정이에요. \
동아리에서 사용중인 프로그램들을 설치하고, 기본적인 환경 설정까지 완료해요.

### 2주차
소프트웨어와 회로를 위주로 사용법을 익혀요. \
소프트웨어는 크게 두 가지를 배우게 돼요.
1. Git과 GitHub
2. 프로그래밍 언어 (C, Python)

하드웨어도 크게 두 가지를 배우게 돼요.
1. 기본적인 회로 이론
2. 회로 도면 그리기

### 3주차
실제로 만들어보는 시간을 가지게 돼요. \
회로의 경우, 2주차에 그린 도면을 실제 회로로 구현해요.

또한 캐드를 사용해서 3D 모델을 제작하고, 3D 프린터를 사용해서 실제로 만들게 돼요.

### 4주차
그동안 배운 것들을 바탕으로, 작은 프로젝트를 해봐요. \
주제는 자율적으로 하되, 몇 가지 기준을 제시해 드릴 예정이에요. \
만약 아이디어가 없으면 예시 주제를 드리니 걱정하지 않아도 돼요.

### 마치며
이렇게 앞으로 4주간 어떤 것을 하게 될지 살펴봤어요. \
지금은 많다고 생각될 수 있지만, 하나씩 하다 보면 금방 해내게 될거에요.
